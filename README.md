## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

```bash
  cp .env.example .env
```

Install dependencies

```bash
  npm install
```

```bash
  composer install
```

Start the server

```bash
  npm run dev
```
```bash
  php artisan migrate
  php artisan serve
```


## Running Tests

To run tests, run the following command

```bash
  php artisan test
```
